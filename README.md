## 📝 Deskripsi Proyek
**MakanYuk** adalah solusi digital untuk pemesanan makanan yang mengedepankan kecepatan dan estetika. Fokus utama pengembangan proyek ini adalah:
1. **Integritas Data**: Mengirimkan informasi pesanan secara akurat dari katalog hingga menjadi struk pembayaran.
2. **Efisiensi Performa**: Penggunaan library Glide untuk manajemen cache gambar dari URL.
3. **Interactive UI**: Penggunaan animasi Lottie untuk meningkatkan pengalaman pengguna (User Experience).

## 🚀 Fitur Utama
- **Interactive Login**: Layar masuk yang interaktif dengan bantuan library **Lottie Animation**.
- **Dynamic Catalog**: Daftar restoran dan menu yang ditampilkan menggunakan **Glide** (Image Loading dari URL).
- **AI Chat Assistant (Simulated)**: Fitur asisten cerdas untuk membantu pengguna mendapatkan rekomendasi promo.
- **Smart Order Handling**: Logika perhitungan harga otomatis (Update Real-time) di halaman Checkout.
- **Digital Receipt System**: Penomoran struk otomatis dan rincian transaksi final di halaman sukses pembayaran.

## 🛠️ Tech Stack & Libraries
- **Language**: Kotlin
- **Layout**: XML (Material Design)
- **Architecture**: ViewBinding
- **Dependencies**:
  - `com.airbnb.android:lottie`: Untuk animasi interaktif.
  - `com.github.bumptech.glide:glide`: Untuk pemrosesan gambar secara asinkron.
  - `de.hdodenhof:circleimageview`: Untuk tampilan foto profil bulat.
  - `androidx.cardview:cardview`: Untuk desain struk dan kontainer menu.
